# 🚀 PIC Microcontroller Projects

A collection of Assembly Language projects developed using the PIC16F627 and PIC16F628 microcontrollers.

These projects demonstrate fundamental embedded systems concepts such as:
- GPIO control
- 7-segment display interfacing
- Push-button input handling
- Software timing delays
- Counter logic using PIC Assembly Language

All programs were written, compiled, and tested using PIC Simulator.

---

# 📂 Projects

## 🔢 Up/Down Counter (0–20)
Microcontroller: PIC16F628

### Features
- Counts from 0 → 20
- Counts from 20 → 0
- Push button changes counting direction
- 7-segment display output
- Software delay routines

### Concepts Demonstrated
- PIC Assembly programming
- Direction control using flags
- Button press detection
- PORTB display interfacing

---

## 🔄 0–9 Reverse Counter
Microcontroller: PIC16F627

### Features
- Counts from 0 → 9
- Automatically reverses from 9 → 0
- 7-segment display output
- Delay loop implementation

### Concepts Demonstrated
- Sequential counting
- Reverse counting
- GPIO configuration
- Delay generation
- Input polling
