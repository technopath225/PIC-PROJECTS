PIC Microcontroller Projects Repository
Overview

This repository contains Assembly Language projects developed using the PIC16F627 Microcontroller and PIC16F628 Microcontroller microcontrollers.
The projects focus on embedded systems concepts such as GPIO control, 7-segment display interfacing, button input handling, and software timing delays.

The code was written, compiled, and tested using PIC Simulator.

Projects Included
1. Up/Down Counter (0–20)

A bidirectional counter built with the PIC16F628 Microcontroller.

Features
Counts from 0 to 20
Counts backward from 20 to 0
Push button changes counting direction
7-segment display output
Delay routines for timing control
Concepts Demonstrated
Assembly programming
Direction control using flags
Button press detection
Display interfacing with PORTB
2. 0–9 Counter with Reverse Counting

A simple automatic counter built with the PIC16F627 Microcontroller.

Features
Counts from 0 to 9
Automatically reverses from 9 back to 0
Displays output on a 7-segment display
Uses software delay loops
Concepts Demonstrated
Sequential counting
Reverse counting
GPIO configuration
Delay generation
Input polling
